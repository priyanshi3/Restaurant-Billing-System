# Restaurant Billing System

The Restaurant Billing System is a WCF-Service-based system that digitally maintains customers and their bills.<br/>
We have made <br/>
        &emsp;- CustomerService (for handling customer-related tasks)<br/>
        &emsp;+ ItemService  (for handling menu items-related tasks)<br/>
        &emsp;+ BillService  (for handling billing tasks)<br/>
        &emsp;+ restaurnatHost<br/> 
        &emsp;+ restaurantClient

## How to run

  First, run restaurantHost and keep it running. <br/>
  Then run restaurantClient and remember the host must be in a running state.
